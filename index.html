Data Cleaning in SQL

Tech layoffs dataset from COVID 2019 were explored and cleaned using SQL.
This project involves analyzing a dataset of tech industry layoffs during the COVID-19 pandemic using SQL. The steps include importing the dataset into SQL Server, performing data cleaning to handle missing values and inconsistencies, and executing SQL queries to explore the data. The analysis focuses on identifying trends such as the number of layoffs over time, affected companies, and geographical distribution, providing insights into the impact of the pandemic on the tech sector.
